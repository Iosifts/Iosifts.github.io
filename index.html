Librosa Cheat Sheet

Section Command Description
Importing Librosa import librosa Imports the librosa library
Loading Audio Files y, sr = librosa.load('filename.wav') Reads and decodes audio files
Playing Audio import soundfile as sf
sf.write('new_filename.wav', y, sr)		 Writes the loaded audio data into a new file
Audio Features
  • Spectrogram: D = librosa.stft(y)
  • Mel Spectrogram: S = librosa.feature.melspectrogram(y, sr=sr)
  • Chroma Feature: C = librosa.feature.chroma_stft(y=y, sr=sr)
  • Spectral Contrast: S_contrast = librosa.feature.spectral_contrast(y=y, sr=sr)
 Computes different features from audio data
Beat and Tempo tempo, beat_frames = librosa.beat.beat_track(y=y, sr=sr) Computes the tempo and beat frames
Display import librosa.display
librosa.display.waveplot(y, sr=sr)		 Displays a waveform plot
Audio Effects
  • HPSS: y_harmonic, y_percussive = librosa.effects.hpss(y)
  • Pitch Shift: y_shifted = librosa.effects.pitch_shift(y, sr, n_steps=4)
 Applies different effects to audio data
Saving Audio librosa.output.write_wav('filename_out.wav', y, sr) Saves the audio data into a .wav file
