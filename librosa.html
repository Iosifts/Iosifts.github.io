Librosa Python Cheat Sheet

This cheat sheet provides a comprehensive collection of useful commands in Librosa, a powerful Python library for audio and music processing. From importing the library to saving your final audio output, this guide covers all the essentials.

1. Importing Librosa

Librosa is a Python library for analyzing and extracting features from music and audio. Start by importing the library:

import librosa

2. Loading Audio Files

Before we can analyze audio, we need to load it into Python. We do this using the load() function, which reads and decodes the audio file. It returns a time series array (y) and its sampling rate (sr).

y, sr = librosa.load('filename.wav')

3. Playing Audio

The soundfile library can be used in conjunction with Librosa to write audio data into a new file, effectively playing it.

import soundfile as sf
sf.write('new_filename.wav', y, sr)

4. Audio Features

4.1 Spectrogram

The short-time Fourier transform can be computed using the stft() function, which is useful for obtaining a spectrogram.

D = librosa.stft(y)

4.2 Mel Spectrogram

The melspectrogram() function computes a Mel-scaled power spectrogram.

S = librosa.feature.melspectrogram(y, sr=sr)

4.3 Chroma Feature

To compute a chromagram from a waveform or spectrogram, use the chroma_stft() function.

C = librosa.feature.chroma_stft(y=y, sr=sr)

4.4 Spectral Contrast

The spectral_contrast() function computes the spectral contrast of the audio.

S_contrast = librosa.feature.spectral_contrast(y=y, sr=sr)

5. Beat and Tempo

The beat_track() function computes the tempo and beat frames of the audio.

tempo, beat_frames = librosa.beat.beat_track(y=y, sr=sr)

6. Display

To display a waveform plot of the audio, use the waveplot() function from librosa.display.

import librosa.display
librosa.display.waveplot(y, sr=sr)

7. Audio Effects

7.1 Harmonic-Percussive Source Separation (HPSS)

The hpss() function separates an audio time-series into its harmonic and percussive components.

y_harmonic, y_percussive = librosa.effects.hpss(y)

7.2 Pitch Shift

To shift the pitch of an audio time-series, use the pitch_shift() function.

y_shifted = librosa.effects.pitch_shift(y, sr, n_steps=4)

8. Saving Audio

The write_wav() function can be used to save the audio time-series data into a .wav file.

librosa.output.write_wav('filename_out.wav', y, sr)

Always consult the Librosa documentation for a more comprehensive guide.
